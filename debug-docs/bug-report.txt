# Bug Report: Advanced Glitch App (Extreme + UI/UX + Chaos + Interaction)

This document lists the intentional bugs implemented in this application for educational purposes.

## Phase 1: The Core Bugs
1.  **Hydration Mismatch**: `app/components/HeroSection.tsx` - Date rendering.
2.  **Improper Storage Access**: `app/components/AuthContext.tsx` - Initializing from window.localStorage.
3.  **State Desync**: `app/components/AuthContext.tsx` - Updates separate from storage.
4.  **Race Condition**: `app/components/GallerySection.tsx` - Async fetches without cancellation.
5.  **Memory Leak**: `app/components/GallerySection.tsx` - Scroll listener without cleanup.
6.  **XSS Vulnerability**: `app/components/CommentsSection.tsx` - `dangerouslySetInnerHTML`.
7.  **Scroll Performance**: `app/components/HeavyScrollComponent.tsx` - Heavy math on scroll.
8.  **Stale Closure**: `app/components/ThemeToggle.tsx` - Dependencies missing in effect.
9.  **Server State Pollution**: `app/utils/singleton-store.ts` - Shared mutable module variable.
10. **CSS Stacking Trap**: `app/components/AdminPanel.tsx` - Invisible overlay blocks button.
11. **Reference Identity Instability**: `app/components/ComplexState.tsx` - Config object recreated every render passed to effect.

## Phase 2: Extreme Mode Bugs
12. **Middleware Redirect Loop**: `middleware.ts` - Logic loop between dashboard and login.
13. **WebSocket Memory Leak**: `app/components/LiveTicker.tsx` - Uncleaned intervals.
14. **Suspense Blocking**: `app/dashboard/page.tsx` - Synchronous work inside Transition.
15. **IndexedDB Conflict**: `app/utils/db.ts` - Connection leaks and upgrade blocking.
16. **Layout Shift**: `app/dashboard/layout.tsx` - Unstable layout.

## Phase 3: UI/UX Nightmares
17. **Body Scroll Lock Fail**: `app/components/ModalTrap.tsx` - Modal fails to unlock body on unmount.
18. **Hover Flicker**: `app/components/FlickeringTooltip.tsx` - Gap between trigger and tooltip.
19. **Ghost Click Area**: `app/components/GhostOverlay.tsx` - Center screen dead zone.

## Phase 4: Chaos Mode
20. **Poltergeist Scroll**: `app/components/PoltergeistScroll.tsx` - Scroll hijack loop.
21. **The Blob**: `app/components/TheBlob.tsx` - Infinite DOM growth.

## Phase 5: Interaction Chaos (New)

### 22. Runaway Button
**Location**: `app/components/RunawayButton.tsx`
**Bug**: `onMouseEnter` handler moves the element to a random position.
**Behavior**: User tries to click the button, but it moves before the click registers.
**Fix Hint**: Remove the evasion logic or use CSS animations that don't affect layout flow if intended (but here it's a bug).

### 23. Input Mirroring (State Pollution)
**Location**: `app/components/InputMirror.tsx`
**Bug**: Two distinct inputs sharing the exact same state variable `[value, setValue]`.
**Behavior**: Typing in "Search" updates "Password" and vice versa.
**Fix Hint**: Use separate state variables for each input.
